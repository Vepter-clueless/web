# 🌐 Výuka Webových technologií (s překvapením)

Tento projekt slouží primárně jako vzdělávací webová platforma pro výuku základů tvorby webových stránek (HTML, CSS a JavaScript). Žáci se zde učí, jak strukturovat obsah, stylovat prvky a přidávat interaktivitu. 

Projekt je ale navržen tak, aby žáky motivoval a bavil – proto **obsahuje masivní, skrytý "Easter Egg"**
