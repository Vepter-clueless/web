Výuka CSS (Kaskádové styly)

0. Co je to CSS?

Zatímco HTML je kostra (říká, co na stránce je), CSS je oblečení (říká, jak to vypadá). Bez CSS by všechny weby vypadaly jako z roku 1995.

<!-- V HTML propojíme CSS pomocí tohoto tagu v hlavičce -->
<link rel="stylesheet" href="style.css">

1. Selektory (Jak vybrat to, co chci obarvit)

Abychom v CSS mohli něčemu změnit barvu, musíme to nejdřív vybrat. Nejčastěji používáme Třídy (Classes). Třída začíná tečkou (.).

Příklad kódu (CSS):
.vip-hrac {
    color: gold;
    font-weight: bold;
}
Příklad kódu (HTML):
<p class="vip-hrac">Tento hráč má zakoupený VIP pass.</p>
Výsledek v prohlížeči:

Tento hráč má zakoupený VIP pass.

2. Barvy a pozadí

Barvy můžeme psát anglicky (např. red, blue) nebo přesně pomocí HEX kódů (např. #ff0000). Pozadí měníme pomocí background-color.

Příklad kódu:
.chybova-hlaska {
    background-color: #ffcccc;
    color: #cc0000;
}
Výsledek v prohlížeči:
ERROR 404: Mozek nenalezen. Zkuste kávu.

3. Písmo a text

Pro změnu písma používáme font-family. Velikost měníme pomocí font-size (často v pixelech - px).

Příklad kódu:
.minecraft-text {
    font-family: 'Courier New', monospace;
    font-size: 24px;
}
Výsledek v prohlížeči:

> Steve joined the game.

4. Box model (Margin a Padding)

Každý prvek na webu je vlastně neviditelná krabice. Padding je vycpávka uvnitř krabice. Margin je mezera venku kolem krabice. Border je rámeček.

Příklad kódu:
.krabice {
    padding: 20px;  /* Vnitřní prostor */
    margin: 10px;   /* Mezera zvenku */
    border: 3px solid blue; /* Modrý rámeček */
}
Výsledek v prohlížeči:
Jsem krabice. Mám kolem sebe vnitřní vycpávku (padding), takže text není nalepený na modré čáře.

5. Hover efekty (Interakce myši)

CSS umí reagovat na to, když uživatel najede myší na prvek. Používáme k tomu tzv. pseudo-třídu :hover. A pokud přidáme transition, změna bude plynulá!

Příklad kódu:
.herni-tlacitko {
    background-color: #5865F2; /* Discord modrá */
    color: white;
    padding: 15px 30px;
    transition: 0.3s; /* Plynulá změna za 0.3 sekundy */
}

.herni-tlacitko:hover {
    background-color: #3ba55d; /* Zelená při najetí */
    transform: scale(1.1); /* Zvětšení o 10% */
}
