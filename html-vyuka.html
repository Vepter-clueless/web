Výuka HTML

1. Úvod do HTML

HTML (HyperText Markup Language) je základní stavební kámen webu. Není to programovací jazyk, ale značkovací jazyk.

Příklad kódu:
<h1>Ahoj světe!</h1> <p>Toto je můj první web.</p>
Výsledek v prohlížeči:

Ahoj světe!

Toto je můj první web.

2. Základní značky

Text formátujeme pomocí párových značek (tagů). Tučný text děláme pomocí tagu <strong> nebo <b>.

Příklad kódu:
<p>Toto je <b>tučný text</b> a toto je <i>kurzíva</i>.</p>
Výsledek v prohlížeči:

Toto je tučný text a toto je kurzíva.

3. Obrázky

Obrázky se vkládají nepárovou značkou <img>. Musíme zadat cestu k obrázku (src) a alternativní popis (alt).

Příklad kódu:
<img src="images.jpg" alt="Ukázkový obrázek">

4. Seznamy

Máme seznamy s odrážkami (UL) a číslované seznamy (OL). Jednotlivé položky značíme LI.

Příklad kódu:
<ul> <li>Mléko</li> <li>Rohlíky</li> </ul>
Výsledek v prohlížeči:
  • Mléko
  • Rohlíky
